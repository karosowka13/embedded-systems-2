About

This is quick programe with basic features of FreeRTOS. You can change parameters in order to see impact on system (i.e. change priority of task of semaphore given by timer). Insight to system is provided by UART and diodies indicating execution of tasks. Try it!

Parameters

Task

Timer

Queue

Configuration

  • set duration, priority of each Task
  • give Semaphore
  • put value to Queue
  • start, stop and change period of software Timer
  • change Semphore given by Timer
